Дополнительные возможности

Такелажный цех

Оснащен современными станками для изготовления всех видов стропов из стального каната и цепей. Оборудование TALURIT (Швеция) для работы со стальным канатом позволяет быстро и точно нарезать канат требуемой длины для различной крановой техники. Для удобства монтажа концы каната оплавляются.

Текстильный цех

Оснащен современным швейным оборудованием. Станком для изготовления круглопрядных стропов грузоподъемностью до 200 тонн.
